Stack In Action

Playwright + axe-core

Automated WCAG checks in UI tests.

import { injectAxe, checkA11y } from 'axe-playwright'; test('checkout is accessible', async ({ page }) => { await page.goto('/checkout'); await injectAxe(page); await checkA11y(page); });

Pa11y CLI

Batch scans for templates and flows.

pa11y https://staging.example.com/checkout --standard WCAG2AA

Lighthouse CI

Automated performance + a11y scoring in CI.

ci: collect: url: ["https://staging.example.com/"]

ARIA Validation

Ensure proper roles and labels in components.

Sample Playwright + axe

import { injectAxe, checkA11y } from 'axe-playwright'; test('checkout is accessible', async ({ page }) => { await page.goto('/checkout'); await injectAxe(page); await checkA11y(page, undefined, { detailedReport: true }); });
