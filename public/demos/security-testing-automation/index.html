Security Testing Automation

Automated security pipeline combining OWASP ZAP, custom scripts, and compliance reporting.

Executive Summary

Challenge

Security checks were inconsistent and findings arrived too late in the release cycle.

Solution

Integrated DAST scans and custom validations into the CI pipeline with clear triage output.

Impact

Shifted security validation left and standardized reporting for remediation tracking.
